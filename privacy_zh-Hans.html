概括

IPcamVision Pro 是一款本地 macOS 应用程序，用于查看、录制和管理用户配置的 IP 摄像机、NVR 和 DVR。该应用程序不会为开发者收集个人数据，不会使用分析、广告跟踪器或分析服务，也不会将您的相机数据发送到我们拥有的任何服务器。设备配置、凭据、录音、快照和偏好设置都会保留在您的 Mac 上，除非您选择导出或共享它们。

由应用程序在本地处理的数据

该应用程序在您的设备上处理以下数据，因为需要它来提供其功能：

  • 摄像机、NVR 和 DVR 地址和配置，包括 IP 地址或主机名、RTSP 或流 URL、端口、流配置文件、PTZ 品牌、网格布局、播放设置和运动检测设置。这些值使用 UserDefaults 存储在应用程序的私有首选项中，并由 macOS 应用程序沙箱隔离。
  • ONVIF 用户名和设备用户名。这些与设备配置一起存储在应用程序的私人首选项中。
  • ONVIF 密码和设备密码。密码存储在应用程序的私有 UserDefaults 区域中，并进行 Base64 编码以避免属性列表序列化问题。 Base64 不是密码加密；它是受 macOS 沙箱保护的应用程序私有存储。如果您需要防止有人物理访问磁盘，请在 Mac 上启用 FileVault。密码不会写入日志，也不会发送给开发人员或第三方。
  • 用户创建的视频录制、下载的播放剪辑和快照。它们仅保存到您通过 macOS 文件选择器选择的文件夹或文件，或者保存到您选择并使用安全范围书签授权的保存文件夹。
  • 本地缩略图、窗口状态、界面首选项、所选语言、购买等级状态和其他应用程序设置。它们保留在 Mac 上的应用程序容器中。

运动检测和通知

运动检测是可选的，并在您的 Mac 上本地运行。该应用程序分析设备上的视频帧（包括可用的 Apple Vision），以对人、车辆、动物或一般运动等运动进行分类。帧不会上传到 Apple、开发人员或任何第三方服务器以实现此功能。

如果您启用运动警报，应用程序可能会向 macOS 请求通知权限。通知由 macOS 发送，可以包括摄像机名称和检测到的主体。您可以在 System Settings 中禁用通知。

该应用程序不做什么

  • 该应用程序不会收集开发者的 IDFA、UDID、MAC 地址或电子邮件地址等标识符。
  • 该应用程序不会向开发人员发送遥测、分析、使用统计或崩溃报告。
  • 该应用程序不使用广告 SDK、分析 SDK 或分析服务。
  • 该应用程序不会将摄像头流、快照、录音、凭证或配置上传到我们拥有的任何服务器。
  • 该应用程序不会跨应用程序或网站跟踪您。
  • 该应用程序不使用Mac相机、麦克风、位置、联系人、日历、蓝牙或照片库权限。

网络通讯

该应用程序仅建立您使用的功能所需的网络连接：

  1. 它直接连接到您在本地网络上或通过 DDNS/端口转发配置的摄像机、NVR 和 DVR。这些连接可以通过 HTTP 或 HTTPS 和 RTSP 流使用 ONVIF。流量直接在您的 Mac 和您配置的设备之间传输。
  2. 当您要求应用程序进行搜索时，它使用 Bonjour 和 WS-Discovery 等本地网络发现技术来查找 LAN 上的兼容设备。
  3. 它联系 Apple 的 StoreKit 服务进行应用内购买和权利检查。开发者不会收到您的 Apple ID、卡号或付款方式。
  4. 如果启用播放时间同步，应用程序可能会通过 NTP 联系 pool.ntp.org，以计算准确 DVR 播放的时钟偏移。此请求不用于跟踪，也不包含您的相机凭据或应用程序配置。
  5. 仅在您按“发送”后，“报告错误”功能才会打开默认邮件客户端并显示一条预先填写的消息。该应用程序本身不上传报告。

运输安全

许多 IP 摄像机和 DVR 仅支持普通 HTTP、RTSP 或自签名 HTTPS 证书。因此，该应用程序允许连接到可能不符合应用程序传输安全默认值的用户配置的相机设备。在您的设备支持的情况下，使用受信任的网络、强相机密码和 HTTPS。

应用内购买

Pro 和 Unlimited 功能通过 Apple 的应用内购买系统使用 StoreKit 解锁。验证通过 Apple 的 API 进行，包括 Transaction.currentEntitlements 和交易更新。该应用程序仅在本地存储生成的权利状态，以便它可以启用购买的功能。

应用程序请求的权限

  • Local Network：macOS 需要发现并连接到本地网络上的摄像机、NVR 和 DVR。您可以在System Settings→Privacy & Security→Local Network中撤销它。
  • 用户选择的文件访问：当您选择保存或导入录音、快照或设备备份文件的位置时使用。如果您选择永久保存文件夹，应用程序会存储该文件夹的安全范围书签。该应用程序无法对磁盘的其余部分进行一般访问。
  • Desktop 或其他文件夹：仅在您明确选择该文件夹作为目标时使用。该应用程序不请求广泛的 Desktop、Documents 或 Downloads 文件夹访问权限。
  • 通知：可选，如果启用，则用于运动警报。
  • Touch ID、Face ID 或 Mac 密码：由 macOS 通过 LocalAuthentication 处理敏感操作，例如泄露相机密码或导出设备数据。该应用程序不接收生物识别数据。

导出、导入和备份配置

该应用程序可以将设备配置导出到您创建和管理的.ipcamvisionprobackup 文件。当前导出省略了单独的 ONVIF 密码字段。但是，如果您手动输入已包含嵌入凭据的 RTSP、DDNS 或流 URL，则该 URL 可以包含在备份中，因为它是设备配置的一部分。如果您导入包含 ONVIF 密码的旧备份，应用程序可能会读取一次并将其迁移到私有应用程序存储中。您负责存储备份文件的位置。

报告错误功能

报告错误窗口不会直接从应用程序发送数据。当您按“发送”时，应用程序将打开您的默认邮件客户端，其中包含一封预填写的电子邮件，地址为 lorys.lg@hotmail.it。您决定是否发送。该电子邮件仅包含您键入的内容以及用于调试的应用程序版本和 macOS 版本。

孩子们

该应用程序面向普通受众。我们不会故意收集有关未成年人的数据，因为该应用程序不会为开发者收集数据。

数据保留和删除

该应用处理的数据将保留在您的 Mac 上，直到您将其删除。您可以从 Settings → Reset data 中删除应用程序数据，从其保存的文件夹中删除生成的录音和快照，或者卸载应用程序。

本政策的变更

未来的更改将在应用程序中发布，并在适用时在规范的在线隐私政策页面上发布。政策更新时，最后更新日期将会更改。
