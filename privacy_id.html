Ringkasan

IPcamVision Pro adalah aplikasi macOS lokal untuk melihat, merekam, dan mengelola kamera IP yang dikonfigurasi pengguna, NVR, dan DVR. Aplikasi ini tidak mengumpulkan data pribadi untuk pengembang, tidak menggunakan analitik, pelacak iklan, atau layanan pembuatan profil, dan tidak mengirimkan data kamera Anda ke server mana pun milik kami. Konfigurasi perangkat, kredensial, rekaman, snapshot, dan preferensi tetap ada di Mac Anda kecuali Anda memilih untuk mengekspor atau membagikannya.

Data ditangani secara lokal oleh aplikasi

Aplikasi ini menangani data berikut di perangkat Anda karena diperlukan untuk menyediakan fitur-fiturnya:

  • Kamera, alamat dan konfigurasi NVR dan DVR, termasuk alamat IP atau nama host, RTSP atau URL streaming, port, profil streaming, merek PTZ, tata letak grid, pengaturan pemutaran, dan pengaturan deteksi gerakan. Nilai-nilai ini disimpan dalam preferensi pribadi aplikasi menggunakan UserDefaults dan diisolasi oleh sandbox aplikasi macOS.
  • Nama pengguna ONVIF dan nama pengguna perangkat. Ini disimpan dengan konfigurasi perangkat di preferensi pribadi aplikasi.
  • Kata sandi ONVIF dan kata sandi perangkat. Kata sandi disimpan di area UserDefaults pribadi aplikasi, dikodekan Base64 untuk menghindari masalah serialisasi daftar properti. Base64 bukan enkripsi kriptografi; ini adalah penyimpanan pribadi aplikasi yang dilindungi oleh kotak pasir macOS. Jika Anda memerlukan perlindungan terhadap seseorang dengan akses fisik ke disk, aktifkan FileVault di Mac Anda. Kata sandi tidak ditulis ke log dan tidak dikirim ke pengembang atau pihak ketiga.
  • Rekaman video, klip pemutaran yang diunduh, dan foto yang dibuat oleh pengguna. Mereka disimpan hanya ke folder atau file yang Anda pilih melalui pemilih file macOS, atau ke folder penyimpanan yang Anda pilih dan otorisasi dengan penanda cakupan keamanan.
  • Thumbnail lokal, status jendela, preferensi antarmuka, bahasa yang dipilih, status tingkat pembelian, dan pengaturan aplikasi lainnya. Ini tetap berada di wadah aplikasi di Mac Anda.

Deteksi gerakan dan pemberitahuan

Deteksi gerakan bersifat opsional dan berjalan secara lokal di Mac Anda. Aplikasi ini menganalisis bingkai video pada perangkat, termasuk dengan Apple Vision jika tersedia, untuk mengklasifikasikan gerakan seperti orang, kendaraan, hewan, atau gerakan umum. Bingkai tidak diunggah ke Apple, ke pengembang, atau ke server pihak ketiga mana pun untuk fitur ini.

Jika Anda mengaktifkan peringatan gerakan, aplikasi mungkin meminta izin notifikasi kepada macOS. Notifikasi dikirimkan oleh macOS dan dapat menyertakan nama kamera dan subjek yang terdeteksi. Anda dapat menonaktifkan notifikasi di System Settings.

Apa yang tidak dilakukan aplikasi ini

  • Aplikasi tidak mengumpulkan pengidentifikasi seperti IDFA, UDID, alamat MAC atau alamat email untuk pengembang.
  • Aplikasi tidak mengirimkan telemetri, analitik, statistik penggunaan, atau laporan kerusakan kepada pengembang.
  • Aplikasi ini tidak menggunakan SDK periklanan, SDK analitik, atau layanan pembuatan profil.
  • Aplikasi tidak mengunggah aliran kamera, snapshot, rekaman, kredensial, atau konfigurasi ke server mana pun milik kami.
  • Aplikasi ini tidak melacak Anda di seluruh aplikasi atau situs web.
  • Aplikasi ini tidak menggunakan izin kamera Mac, mikrofon, lokasi, kontak, kalender, Bluetooth, atau perpustakaan Foto.

Komunikasi jaringan

Aplikasi ini hanya membuat koneksi jaringan yang diperlukan untuk fitur yang Anda gunakan:

  1. Ini terhubung langsung ke kamera, NVR dan DVR yang Anda konfigurasikan, baik di jaringan lokal Anda atau melalui DDNS / penerusan port. Koneksi ini dapat menggunakan ONVIF melalui aliran HTTP atau HTTPS dan RTSP. Lalu lintas langsung mengalir antara Mac Anda dan perangkat yang Anda konfigurasikan.
  2. Ia menggunakan teknologi penemuan jaringan lokal seperti Bonjour dan WS-Discovery untuk menemukan perangkat yang kompatibel di LAN Anda saat Anda meminta aplikasi untuk mencari.
  3. Ini menghubungi layanan StoreKit Apple untuk pembelian dalam aplikasi dan pemeriksaan hak. Pengembang tidak menerima ID Apple, nomor kartu, atau metode pembayaran Anda.
  4. Jika sinkronisasi waktu pemutaran diaktifkan, aplikasi dapat menghubungi pool.ntp.org melalui NTP untuk menghitung offset jam untuk pemutaran DVR yang akurat. Permintaan ini tidak digunakan untuk pelacakan dan tidak menyertakan kredensial kamera atau konfigurasi aplikasi Anda.
  5. Fitur Laporkan bug membuka klien email default Anda dengan pesan yang sudah diisi sebelumnya hanya setelah Anda menekan Kirim. Aplikasi itu sendiri tidak mengunggah laporan.

Keamanan transportasi

Banyak kamera IP dan DVR hanya mendukung sertifikat HTTP, RTSP, atau HTTPS yang ditandatangani sendiri. Oleh karena itu, aplikasi mengizinkan koneksi ke perangkat kamera yang dikonfigurasi pengguna yang mungkin tidak memenuhi default Keamanan Transportasi Aplikasi. Gunakan jaringan tepercaya, kata sandi kamera yang kuat, dan HTTPS jika perangkat Anda mendukungnya.

Pembelian dalam aplikasi

Fitur Pro dan Unlimited dibuka melalui sistem pembelian dalam aplikasi Apple menggunakan StoreKit. Verifikasi terjadi melalui API Apple, termasuk Transaction.currentEntitlements dan pembaruan transaksi. Aplikasi hanya menyimpan status kepemilikan yang dihasilkan secara lokal sehingga dapat mengaktifkan fitur yang dibeli.

Izin diminta oleh aplikasi

  • Local Network: diperlukan oleh macOS untuk menemukan dan terhubung ke kamera, NVR, dan DVR di jaringan lokal Anda. Anda dapat mencabutnya di System Settings → Privacy & Security → Local Network.
  • Akses file yang dipilih pengguna: digunakan saat Anda memilih tempat untuk menyimpan atau mengimpor rekaman, snapshot, atau file cadangan perangkat. Jika Anda memilih folder penyimpanan persisten, aplikasi akan menyimpan bookmark dengan cakupan keamanan untuk folder tersebut. Aplikasi ini tidak mendapatkan akses umum ke seluruh disk Anda.
  • Desktop atau folder lain: hanya digunakan ketika Anda secara eksplisit memilih folder tersebut sebagai tujuan. Aplikasi tidak meminta akses folder Desktop, Documents, atau Downloads yang luas.
  • Notifikasi: opsional, digunakan untuk peringatan gerakan jika Anda mengaktifkannya.
  • Kata sandi Touch ID, Face ID atau Mac: ditangani oleh macOS melalui LocalAuthentication untuk tindakan sensitif seperti mengungkapkan kata sandi kamera atau mengekspor data perangkat. Aplikasi tidak menerima data biometrik.

Mengekspor, mengimpor, dan mencadangkan konfigurasi

Aplikasi ini dapat mengekspor konfigurasi perangkat ke file.ipcamvisionprobackup yang Anda buat dan kelola. Ekspor saat ini menghilangkan bidang kata sandi ONVIF yang terpisah. Namun, jika Anda memasukkan RTSP, DDNS, atau URL streaming secara manual yang sudah berisi kredensial tertanam, URL tersebut dapat disertakan dalam cadangan karena merupakan bagian dari konfigurasi perangkat. Jika Anda mengimpor cadangan lama yang berisi kata sandi ONVIF, aplikasi mungkin membacanya satu kali untuk memigrasikannya ke penyimpanan aplikasi pribadi. Anda bertanggung jawab atas tempat Anda menyimpan file cadangan.

Laporkan fitur bug

Jendela Laporkan bug tidak mengirim data langsung dari aplikasi. Saat Anda menekan Kirim, aplikasi akan membuka klien email default Anda dengan email yang telah diisi sebelumnya yang dialamatkan ke LorysMay@outlook.it. Anda memutuskan apakah akan mengirimkannya. Email hanya berisi apa yang Anda ketik ditambah versi aplikasi dan versi macOS untuk debugging.

Anak-anak

Aplikasi ini ditujukan untuk khalayak umum. Kami tidak dengan sengaja mengumpulkan data tentang anak di bawah umur karena aplikasi tidak mengumpulkan data untuk pengembang.

Retensi dan penghapusan data

Data yang ditangani oleh app tetap ada di Mac Anda hingga Anda menghapusnya. Anda dapat menghapus data aplikasi dari Settings → Reset data, menghapus rekaman dan snapshot yang dihasilkan dari folder tersimpannya, atau menghapus instalan aplikasi.

Perubahan pada kebijakan ini
