In breve

IPcamVision Pro è un'app locale. Non raccogliamo alcun dato personale, non inviamo i tuoi dati a nessun server e non utilizziamo analytics, tracker pubblicitari o servizi di profilazione. Tutto ciò che configuri (telecamere, credenziali, registrazioni, snapshot) resta esclusivamente sul tuo Mac.

Dati gestiti dall'app

L'app gestisce sul tuo dispositivo i seguenti dati, necessari per il suo funzionamento:

  • Indirizzi e configurazione delle tue telecamere/NVR/DVR (IP o hostname, porte, profili stream, brand PTZ, opzioni). Salvati in modo non cifrato nelle preferenze dell'app (UserDefaults), perché non sono informazioni segrete.
  • Nome utente delle telecamere ONVIF. Salvato come sopra.
  • Password delle telecamere ONVIF. Salvate nel Portachiavi (Keychain) di macOS, protetto dal sistema operativo. Le password non vengono mai scritte in chiaro nei file di preferenze, nei log, né condivise con noi o con terze parti.
  • Registrazioni video e snapshot generati dall'utente. Salvati nella cartella che l'utente sceglie. Non vengono caricati da nessuna parte.
  • Preferenze d'interfaccia (layout griglia, dimensioni finestra, scorciatoie). Salvate nelle preferenze dell'app.

Cosa l'app NON fa

  • Non raccoglie identificatori (UDID, IDFA, indirizzo MAC, e-mail).
  • Non invia telemetria, statistiche d'uso, crash report a noi o a terzi.
  • Non utilizza SDK di analytics (es. Google Analytics, Firebase, Mixpanel).
  • Non utilizza SDK pubblicitari.
  • Non profila l'utente in alcun modo.
  • Non comunica con nessun server di nostra proprietà: non ne abbiamo.

Comunicazioni di rete

L'app si collega solo a:

  1. Le telecamere, gli NVR e i DVR che tu configuri, sulla rete locale o tramite DDNS/port-forwarding sulla rete pubblica. Queste connessioni usano i protocolli ONVIF (HTTP/HTTPS) e RTSP, con le credenziali che hai inserito tu. Il traffico va direttamente dalla tua app alla telecamera, senza passare da nessun nostro server.
  2. Il servizio In-App Purchase di Apple, gestito interamente dal sistema operativo per la gestione degli acquisti delle funzioni Pro. Non riceviamo alcuna informazione personale tua: Apple ci comunica solo che un certo identificatore di acquisto è stato attivato.

Acquisti in-app

Le funzioni Pro vengono sbloccate tramite il sistema di acquisto in-app di Apple (StoreKit). La verifica avviene localmente sul tuo dispositivo attraverso le API di Apple (Transaction.currentEntitlements); non manteniamo né riceviamo informazioni sul tuo Apple ID, sulla tua carta o sul tuo metodo di pagamento.

Permessi richiesti dall'app

  • Rete locale: richiesto da macOS per la scoperta delle telecamere tramite ONVIF/WS-Discovery e Bonjour sulla LAN. Lo concedi tu al primo uso e puoi revocarlo da Impostazioni di sistema → Privacy e sicurezza → Rete locale.
  • Accesso a file scelti dall'utente: l'app può salvare registrazioni e snapshot solo nella cartella che selezioni esplicitamente tramite il pannello di sistema. Non ha accesso al resto del disco.

Esportazione e backup delle configurazioni

L'app ti consente di esportare le configurazioni delle telecamere in un file .ipcamvisionprobackup. Per impostazione predefinita questo file non contiene le password ONVIF (che vivono solo nel Keychain del tuo Mac). Se importi un file di backup creato con versioni precedenti dell'app che contengono password in chiaro, l'app sposta automaticamente quelle password nel Keychain al primo avvio. Il file di backup resta comunque un file generato e gestito da te: sei responsabile del luogo in cui lo conservi.

Funzione "Segnala bug"

Dall'app puoi aprire una finestra di segnalazione bug. Quando premi "Invia", l'app non invia nulla via rete: apre il tuo client di posta predefinito con un'email precompilata indirizzata a lorys.lg@hotmail.it. Sei tu a decidere se inviarla o meno. L'email contiene solo ciò che hai scritto, più la versione dell'app e la versione di macOS (informazione utile per il debug).

Minori

L'app è destinata a un pubblico adulto generico. Non raccoglie intenzionalmente dati di minori perché non raccoglie dati di nessuno.

Conservazione dei dati

Tutti i dati gestiti dall'app risiedono esclusivamente sul tuo Mac. Puoi cancellarli in qualsiasi momento dal pannello "Impostazioni → Reset dati" dell'app, oppure disinstallando l'app. La cancellazione include anche le voci salvate nel Portachiavi.
