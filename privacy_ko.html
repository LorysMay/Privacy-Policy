요약

IPcamVision Pro는 사용자 구성 IP 카메라, NVR 및 DVR를 보고, 기록하고, 관리하기 위한 로컬 macOS 앱입니다. 이 앱은 개발자를 위해 개인 데이터를 수집하지 않으며 분석, 광고 추적기 또는 프로파일링 서비스를 사용하지 않으며 카메라 데이터를 당사가 소유한 서버로 보내지 않습니다. 장치 구성, 자격 증명, 녹음, 스냅샷 및 환경 설정은 내보내거나 공유하도록 선택하지 않는 한 Mac에 그대로 유지됩니다.

앱에서 로컬로 처리되는 데이터

앱은 기능을 제공하는 데 필요하므로 기기에서 다음 데이터를 처리합니다.

  • IP 주소 또는 호스트 이름, RTSP 또는 스트림 URL, 포트, 스트림 프로필, PTZ 브랜드, 그리드 레이아웃, 재생 설정 및 동작 감지 설정을 포함한 카메라, NVR 및 DVR 주소 및 구성. 이러한 값은 UserDefaults를 사용하여 앱의 개인 기본 설정에 저장되며 macOS 앱 샌드박스에 의해 격리됩니다.
  • ONVIF 사용자 이름 및 장치 사용자 이름. 이는 앱의 개인 기본 설정에 있는 기기 구성과 함께 저장됩니다.
  • ONVIF 비밀번호 및 장치 비밀번호. 비밀번호는 macOS 키체인에 저장된 마스터 키를 사용하여 AES-256-GCM으로 암호화된 후 앱의 비공개 UserDefaults 영역에 저장됩니다. 마스터 키는 한 번만 생성되며 절대 키체인 밖으로 나가지 않습니다. 비밀번호는 디스크에 평문이나 Base64 형식으로 저장되지 않습니다. 비밀번호는 로그에 기록되지 않으며 개발자나 제3자에게 전송되지 않습니다.
  • 사용자가 생성한 비디오 녹화, 다운로드한 재생 클립 및 스냅샷. macOS 파일 선택기를 통해 선택한 폴더나 파일에만 저장되거나, 보안 범위 북마크로 선택하고 인증한 저장 폴더에만 저장됩니다.
  • 로컬 썸네일, 창 상태, 인터페이스 기본 설정, 선택한 언어, 구매 계층 상태 및 기타 앱 설정. 이는 Mac의 앱 컨테이너에 남아 있습니다.

모션 감지 및 알림

동작 감지는 선택 사항이며 Mac에서 로컬로 실행됩니다. 앱은 Apple Vision을 사용할 수 있는 경우 이를 포함하여 기기에서 비디오 프레임을 분석해 사람, 차량, 동물 또는 일반 움직임으로 분류합니다. 이 로컬 분석은 프레임을 Apple, 개발자 또는 개발자가 소유한 서버로 업로드하지 않습니다.

Telegram 동작 알림을 활성화하면 앱은 알림 텍스트와, 사진 옵션을 켠 경우에만 동작 스냅샷을 api.telegram.org의 Telegram Bot API로 전송합니다. Telegram은 Telegram이 운영하며 개발자가 소유하거나 제어하는 서버가 아닙니다. Telegram 사용 여부, 스냅샷 포함 여부, 앱 설정의 자동 삭제 타이머 설정 여부는 사용자가 결정합니다. Telegram의 전달 및 보관은 Telegram 정책을 따릅니다. 자동 삭제는 Mac이 켜져 있고 앱이 실행 중일 때만 수행될 수 있으며 Telegram의 삭제 제한을 따릅니다.

Telegram 알림을 사용하려면 사용자가 직접 Telegram 봇을 만들거나 설정하고 앱에 봇 토큰과 Chat ID를 입력합니다. 봇 토큰과 Chat ID는 Mac에 로컬로 저장되며 개발자에게 전송되지 않고, Telegram 알림이 활성화된 경우 Telegram Bot API에 연결하는 데만 사용됩니다.

macOS 동작 알림을 활성화하면 앱이 macOS 알림 권한을 요청할 수 있습니다. 로컬 알림은 macOS가 전달하며 카메라 이름과 감지된 대상을 포함할 수 있습니다. 시스템 설정에서 끌 수 있습니다.

앱이 하지 않는 일

  • 앱은 개발자를 위해 IDFA, UDID, MAC 주소 또는 이메일 주소와 같은 식별자를 수집하지 않습니다.
  • 앱은 원격 측정, 분석, 사용 통계 또는 충돌 보고서를 개발자에게 보내지 않습니다.
  • 앱은 광고 SDK, 분석 SDK 또는 프로파일링 서비스를 사용하지 않습니다.
  • 앱은 카메라 스트림, 스냅샷, 녹화, 자격 증명 또는 구성을 당사가 소유한 서버에 업로드하지 않습니다.
  • 앱은 앱이나 웹사이트 전반에서 사용자를 추적하지 않습니다.
  • 앱은 Mac 카메라, 위치, 연락처, 캘린더, Bluetooth 또는 사진 보관함 권한을 사용하지 않습니다.

네트워크 통신

앱은 귀하가 사용하는 기능에 필요한 네트워크 연결만 수행합니다.

  1. 로컬 네트워크에서 또는 DDNS/포트 전달을 통해 구성한 카메라, NVR 및 DVR에 직접 연결됩니다. 이러한 연결은 HTTP 또는 HTTPS 및 RTSP 스트림을 통해 ONVIF를 사용할 수 있습니다. Mac과 구성한 장치 간에 트래픽이 직접 이동합니다.
  2. 앱에 검색을 요청할 때 Bonjour 및 WS-Discovery와 같은 로컬 네트워크 검색 기술을 사용하여 LAN에서 호환되는 장치를 찾습니다.
  3. 인앱 구매 및 자격 확인을 위해 Apple의 StoreKit 서비스에 연결됩니다. 개발자는 귀하의 Apple ID, 카드 번호 또는 결제 방법을 받지 않습니다.
  4. 재생 시간 동기화가 활성화된 경우 앱은 정확한 DVR 재생을 위해 클럭 오프셋을 계산하기 위해 NTP를 통해 pool.ntp.org에 연결할 수 있습니다. 이 요청은 추적에 사용되지 않으며 카메라 자격 증명이나 앱 구성을 포함하지 않습니다.
  5. Telegram 동작 알림을 활성화하면 앱은 HTTPS를 통해 api.telegram.org의 Telegram Bot API에 연결하여 알림 텍스트와, 사진 알림을 켠 경우에만 선택한 동작 스냅샷을 전달합니다. Telegram은 개발자가 아닌 Telegram이 운영하는 타사 서비스입니다.
  6. 버그 신고 기능은 보내기를 누른 후에만 미리 채워진 메시지가 포함된 기본 메일 클라이언트를 엽니다. 앱 자체에서는 보고서를 업로드하지 않습니다.

운송 보안

많은 IP 카메라와 DVR는 일반 HTTP, RTSP 또는 자체 서명된 HTTPS 인증서만 지원합니다. 이러한 이유로 앱은 앱 전송 보안 기본값을 충족하지 않을 수 있는 사용자 구성 카메라 장치에 대한 연결을 허용합니다. 장치가 지원하는 경우 신뢰할 수 있는 네트워크, 강력한 카메라 비밀번호 및 HTTPS를 사용하세요.

인앱 구매

Pro 및 Unlimited 기능은 StoreKit를 사용하는 Apple의 인앱 구매 시스템을 통해 잠금 해제됩니다. 확인은 Transaction.currentEntitlements 및 거래 업데이트를 포함한 Apple의 API를 통해 이루어집니다. 앱은 구매한 기능을 활성화할 수 있도록 결과 자격 상태만 로컬에 저장합니다.

앱에서 요청하는 권한

  • Local Network: macOS가 로컬 네트워크에서 카메라, NVR 및 DVR를 검색하고 연결하는 데 필요합니다. System Settings → Privacy & Security → Local Network에서 취소할 수 있습니다.
  • 사용자 선택 파일 액세스: 녹화, 스냅샷 또는 장치 백업 파일을 저장하거나 가져올 위치를 선택할 때 사용됩니다. 영구 저장 폴더를 선택하면 앱은 해당 폴더에 대한 보안 범위 북마크를 저장합니다. 앱은 디스크의 나머지 부분에 대한 일반적인 액세스 권한을 얻지 못합니다.
  • Desktop 또는 기타 폴더: 해당 폴더를 대상으로 명시적으로 선택한 경우에만 사용됩니다. 앱은 광범위한 Desktop, Documents 또는 Downloads 폴더 액세스를 요청하지 않습니다.
  • 알림: 선택 사항이며 활성화한 경우 동작 알림에 사용됩니다.
  • 마이크: 선택 사항이며 호환 카메라로 양방향 통화 / talkback을 시작할 때만 사용됩니다. 시스템 설정에서 거부하거나 취소할 수 있습니다.
  • Touch ID, Face ID 또는 Mac 비밀번호: 카메라 비밀번호 공개 또는 장치 데이터 내보내기와 같은 민감한 작업을 위해 macOS에서 LocalAuthentication까지 처리됩니다. 앱은 생체 인식 데이터를 수신하지 않습니다.

구성 내보내기, 가져오기 및 백업

앱은 장치 구성을 사용자가 만들고 관리하는.ipcamvisionprobackup 파일로 내보낼 수 있습니다. 현재 내보내기에서는 별도의 ONVIF 비밀번호 필드가 생략됩니다. 그러나 이미 내장된 자격 증명이 포함된 RTSP, DDNS 또는 스트림 URL을 수동으로 입력한 경우 해당 URL은 장치 구성의 일부이므로 백업에 포함될 수 있습니다. ONVIF 비밀번호가 포함된 이전 백업을 가져오는 경우 앱은 이를 한 번 읽고 개인 앱 저장소로 마이그레이션할 수 있습니다. 백업 파일을 저장하는 위치에 대한 책임은 귀하에게 있습니다.

버그 신고 기능

버그 신고 창은 앱에서 직접 데이터를 전송하지 않습니다. 보내기를 누르면 앱에서 LorysMay@outlook.it라는 주소가 미리 입력된 이메일이 포함된 기본 메일 클라이언트를 엽니다. 보낼지 여부를 결정합니다. 이메일에는 입력한 내용과 디버깅을 위한 앱 버전 및 macOS 버전만 포함됩니다.

어린이들

이 앱은 일반 사용자를 대상으로 합니다. 앱은 개발자를 위해 데이터를 수집하지 않기 때문에 미성년자에 대한 데이터를 고의로 수집하지 않습니다.

데이터 보관 및 삭제

앱에서 처리하는 데이터는 삭제할 때까지 Mac에 남아 있습니다. Settings → Reset data에서 앱 데이터를 제거하거나, 저장된 폴더에서 생성된 녹음 및 스냅샷을 삭제하거나, 앱을 제거할 수 있습니다.
